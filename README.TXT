Photo Scenery Maker : Source files

-- Compile

You need Borland C++ Builder 6 to compile.
Open PhotoSceneryMaker.bpr file then compile it.

-- Translation

You can translate all messages to another language.

You need 'GNU gettext for Delph, C++ Builder and Kylix'.

Create locale/[language]/LC_MESSAGE dir, copy default.po file 
to the dir, translate it, then create .mo file.

For more information, see http://dybdahl.dk/dxgettext/.
