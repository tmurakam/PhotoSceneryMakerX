Generate scenery

Select "Scenery Generation" menu.

Scenery generation process consists of following steps:

  1. Create inf file for resample.exe.
  2. Execute resample.exe of Terrain SDK. Bitmap files and tmf files are created in this process.
  3. Merge alpha texture. In this process, textures and alpha textures are merged into targa (.tga) format bitmaps.
  4. Convert texture. In this process, targa format bitmaps are converted into BMP files used in Flight Simulator.
  5. Generate BGL file. Tmf files are compressed and BGL file is generated from the tmf file.
Usually use "Run Whole Process".

That's all!!! All scenery files are generated.
