シーナリ生成

シーナリを生成します。「シーナリ生成」メニューを選ぶと以下のダイアログが表示されます。

シーナリの生成は２つのステップに分かれています。

  1. inf ファイルの生成。このステップは、Terrain SDK の resample.exe に入力するための inf ファイルを生成します。
  2. Terrain SDK の resample.exe 実行。このステップで、シーナリ(BGLファイル)が生成されます。
通常は「全実行」を使用してください。上記の全てのステップが全て実行されます。

各ステップを別々に行いたい場合は、画面右側のボタン群を使用してください。

これでシーナリの生成は完了です。
