シーナリ生成

シーナリを生成します。「シーナリ生成」メニューを選ぶと以下のダイアログが表示されます。

シーナリの生成は５つのステップに分かれています。

  1. inf ファイルの生成。このステップは、Terrain SDK の resample.exe に入力するための inf ファイルを生成します。
  2. Terrain SDK の resample.exe 実行。このステップで、ビットマップファイルの分割と tmf ファイルの生成が行われます。
  3. アルファテクスチャのマージ。このステップでは、通常テクスチャとアルファテクスチャを Targa (.tga) 形式のファイルにマージします。
  4. テクスチャ変換。このステップでは、Imagetool ユーティリティを使って、Targa ファイルを Flight Simulator が利用する bmp ファイルに変換します。
  5. BGL ファイルの生成。Terrain SDK を使って、tmf ファイルの圧縮と BGL ファイルへの変換処理を行います。
通常は「全実行」を使用してください。上記の全てのステップが全て実行されます。

各ステップを別々に行いたい場合は、画面右側のボタン群を使用してください。

これでシーナリの生成は完了です。
