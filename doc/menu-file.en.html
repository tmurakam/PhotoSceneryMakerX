File menu

New

Create new project.

Open project.

Save current project.

Save As

Save current project as another file name.

Project property

Open project property dialog.

Bitmaps tab

  • Support season : Check if seasonal/lightmap bitmap are needed.
  • Summer/Spring/Fall/Winter/Harsh Winter : Specify seasonal bitmap files.
  • Lightmap : Specify lightmap bitmap file.
  • Water (alpha) : Specify water (alpha) bitmap file (optional).

Coordinates tab

  • Bitmap Boundary : Specify bitmap boundary latitude/longitude. You can use Calibration : 2 point calibration to specify them.
  • Resolution : Specify bitmap resolution in degree/pixel. You can use Calibration : 2 point calibration to specify them.
  • Info : Resolution in m/pixel, bitmap width/height are displayed.

Output tab

  • File output
    • Output Folder : Specify scenery output folder.
    • Base File Name : Specify scenery BGL file base name.
  • Bitmap boundary
    • Use whole bitmap : If checked, whole bitmap area are used to generate scenery.
    • Top/Left/Right/Bottom : You can specify bitmap area to be used. You can use Calibration : bitmap boundary to specify them.
