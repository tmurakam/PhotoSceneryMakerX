ビットマップタブ

ここでは、使用するビットマップファイルを指定します。 ビットマップファイルは１４個まで指定でき、それぞれに種別（日中/夜間/水面マスク）や使用する月の指定が可能です。

なお、ビットマップは最低でも１個は指定が必要です。Photo Scenery Maker の画面に表示されるのは１番のビットマップだけです。

  • 「ビットマップ番号」: ビットマップの番号を指定します。
  • 「ビットマップファイル」: 各番号に対応したビットマップファイル(*.bmp)を指定します。
  • 「種別」:ここでは、ビットマップの種別を指定します。
    • 「すべて」：日中、夜間両方の状況でこのビットマップを使用します。
    • 「日中」：日中のみ、このビットマップを使用します。
    • 「夜間」：夜間にのみ、このビットマップを使用します。
    • 「地表/水面マスク」：水面を指定するためのビットマップ（アルファチャネル）として使用します。
    • 「月の指定」：このビットマップを通年で使用するのか、特定の月に使用するのかを指定します。この指定は、種別を「日中」にした場合にのみ意味を持ちます。

座標タブ

  • ビットマップ範囲 : ビットマップがカバーするエリアの緯度/経度範囲を指定します。 補正 : ２点補正を行えば、自動的に入力されます。
  • 解像度 : ビットマップの解像度 (秒/pixel) を指定します。 補正 : ２点補正を行えば、自動的に入力されます。
  • 情報 : m/pixel 単位の解像度、ビットマップのサイズが表示されます。

出力タブ

  • ファイル出力
    • 出力フォルダ : シーナリファイルの生成先フォルダを指定します。
    • ベースファイル名 : シーナリのベースファイル名を指定します。BGL ファイルのファイル名 (拡張子除く）になります。
  • ビットマップ範囲
    • ビットマップ全体を使用 : チェックすると、自動的にビットマップ全体をシーナリに 使用します。チェックを外すと、以下で指定する範囲だけが使用されます。
    • 上/左/右/下 : 使用するビットマップの範囲を、ビットマップ座標で指定します。 補正 : ビットマップ範囲 で指定することもできます。
  • シーナリ解像度
    • シーナリの解像度(LOD)を指定します。通常は -1 としておきます(自動設定)。 13 に設定すると 4.7m/pixel になります。LOD 値を１つ増やすたびに、解像度が倍になりますが、 その分生成されるシーナリは大きくなります。
