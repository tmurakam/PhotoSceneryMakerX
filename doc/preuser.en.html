For users of previous version of Photo Scenery Maker

Photo Scenery maker for FSX is the tool for FSX (Flight Simulator X) only. It can't create scenery for FS2002/2004. To create FS2002/2004 scenery, use older (not for FSX) version.

Some differences exist compared to previous version:

  • New project files (*.psmx) are not compatible with older format (*.psm). Older format can be loaded, but some information will lack.

  • The method to specify bitmap files are totally different.
    This version can specify up to 14 bitmaps, because you can specify 'month' for each bitmaps with FSX.

  • You can specify scenery resolution.
