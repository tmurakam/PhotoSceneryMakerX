Setup

Setup FSX SDK

Install FSX SDK from SDK folder of FSX DVD ROM #1.

Note: Standard edition does not contain FSX SDK. Please purchase Deluxe version.

Setup option

Start up the Photo Scenery Maker, select "Option" menu, and setup the options.

  • "FSX SDK folder" : Specify Terrain SDK folder of FSX SDK. By default, it is "C:\Program Files\Microsoft Games\Microsoft Flight Simulator X SDK\SDK\Environment Kit\Terrain SDK".
  • "Language" : Specify your favorite language.
