セットアップ

FSX SDKのセットアップ

まず、FSX SDK をセットアップしてください。FSX SDK は、FSX の DVD-ROM 1枚目の SDK フォルダにあります。

注: 英語版の Standard Edition には FSX SDK は含まれていません。

オプションの設定

Photo Scenery Maker を起動し、「Options」を選び、オプションを設定します。

  • "FSX SDK folder" には、FSX SDK に含まれる Terrain SDK のフォルダを指定します。 デフォルトでは、 "C:\Program Files\Microsoft Games\Microsoft Flight Simulator X SDK\SDK\Environment Kit\Terrain SDK" です。
  • "Language" には Photo Scenery Maker が表示に使用する言語を設定します。
