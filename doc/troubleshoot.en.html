Texture files are not produced correctly...

First, please check that you are using correct version of resample.exe. Resample.exe with FS2002 Custom Terrain SDK DOES NOT WORK!. You MUST use FS2000 Terrain SDK!!!

Next, please check if the coordinates parameters are valid. Almost all troubles are caused by this problem. Photo Scenery Maker ver 1.2.2 checks these parameters before generate scenery.
